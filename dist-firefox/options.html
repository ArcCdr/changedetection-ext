Setup Instructions

  1. Make sure your changedetection.io server is running and accessible
  2. Log into your changedetection.io web interface
  3. Go to Settings and enable the API
  4. Copy your API key and paste it above
  5. Enter your server URL (e.g., http://localhost:5000)
  6. Click "Test Connection" to verify everything works
  7. Click "Save Settings" to store your configuration

API Documentation

This extension uses the changedetection.io REST API. For more information, visit the API documentation at your server's /api/docs endpoint.
